### 后端数据流（一次分析的完整路径）

```
用户粘贴文字 → /analyze路由
  → 检查次数(UID+IP双轨)
  → 智能分段(判例) → 插入[第X段]编号
  → ThreadPoolExecutor并行4路 → 问AI(提示词+编号文本)
  → 案例总结(依赖上面4个结果)
  → 溯源对比(原文 vs AI引用)
  → 法条库查询
  → 验证分析结果
  → 存储为JSON
  → return jsonify({...分析,溯源,法条,验证,剩余次数})
```
