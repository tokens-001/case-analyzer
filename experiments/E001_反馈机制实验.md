# E001 — 反馈机制实验

## 假设

选择题反馈的完成率远高于开放文本反馈。用户不是产品经理，不会写长反馈。

## 来源

判例助手 v3

## 实验设计

- A 组（旧）：结果页底部两个按钮 👍 有帮助 / 👎 有问题
- B 组（新）：5 步结构化选择题（有帮助程度 + 最有价值模块 + 最没用模块 + 问题类型 + 选填文字）
- 衡量指标：反馈提交率、反馈数据可用性

## 结果

- A 组：零反馈（按钮存在但无人点击）
- B 组：待真实用户数据验证（结构设计完成，已部署）

## 状态：🔄 进行中（等待用户数据）

## 关联

- learning: 003_选择题反馈优于开放文本
- pattern: P002_选择题反馈优先原则（等待本实验 + 1 个额外项目验证）
