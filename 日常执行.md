# 日常执行

原则：一天精力池只够装一样重脑力。Python和读书不分同一天。思考案卷随时写，不受限制。体系维护触发了就处理。

## 七日循环
| 第1天 | 第2天 | 第3天 | 第4天 | 第5天 | 第6天 | 第7天 |
|-------|-------|-------|-------|-------|-------|-------|
| Python | Python | 读书+电影 | Python | Python | 读书+电影 | 总复习 |

- Python日：学新内容前先复习10分钟。学新语法+动手实践
- 读书+电影日：感性低迷时读最佳；状态不对可换Python。连续两次未读→第三次必须读
- 总复习日：不看文件，看速查卡提示，把本周所有新代码完整重写一遍。卡住了翻文件，但不能抄

## 每周
- AI法学习（每周拆解一篇判例或论文）
- 作品账户（每周至少一篇公开内容）

## 每日闪卡（练完金刚功后，1-2分钟）
- 过一遍昨天的阅读笔记（一句话）
- 随机抽一张速查卡读一遍（CLAUDE.md 触发对照表）
- 不测试不背诵，纯过目


## 长期·每日
- 完善知识案卷（随时）
- 八部金刚功：每周至少三次，哪天做哪天打勾。不打勾不自责 (6/4 ✅)
- 晚上练腹肌（平板支撑，2组力竭）
- 出行自行车 (6/4 ✅)
