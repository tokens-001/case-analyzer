# 日常执行

原则：一天精力池只够装一样重脑力。Python和读书不分同一天。思考案卷随时写，不受限制。体系维护触发了就处理。

## 周循环（每周六读书，周日复习，其余Python）
| 一 | 二 | 三 | 四 | 五 | 六 | 日 |
|---|---|---|---|---|---|---|
| Python | Python | Python | Python | Python | 读书+电影 | 总复习 |

- Python日：学新前先复习10分钟
- 周六读书：感性低迷时读最佳。状态不对可换Python
- 周日总复习：看速查卡提示，完整重写本周所有代码。卡住翻文件，不能抄
- 身体锻炼（金刚功+腹肌）：放在读书日和复习日
- 自行车：外出就骑，不追踪
- 每周：AI法拆解一篇判例/论文，作品账户一篇公开内容
