# Python学习案卷

## 第一节：第一句话
- `print("内容")` — 把括号里的东西输出到屏幕
- `"内容"` 叫字符串，用引号包起来

## 第二节：让程序问问题
- `input("提示文字")` — 输出提示，停下来等人输入，返回输入的内容
- 变量 — 把数据存起来，比如 `name = input("你是谁？")`，之后用 `name` 就能拿到输入的内容
- `+` — 把字符串拼在一起

## 第三节：判断
- `if 条件:` — 如果条件成立，执行下面缩进的代码
- `elif 条件:` — 否则如果另一个条件成立
- `else:` — 以上都不成立时执行
- `==` — 判断两边是否相等（一个 `=` 是赋值，两个 `==` 才是比较）
- **缩进必须一致**，Python 靠缩进识别代码块
- ✅ 已实践：guess.py 三次测试通过

## 第三节实践补充：input()
- `name = input("提示")` — 把用户输入的内容存到变量 name 里
- `print("你好，" + name)` — 用 + 拼接字符串和变量
- ✅ 已实践：ask.py 测试通过

## 第四节：循环
- `while 条件:` — 条件成立就一直重复执行缩进的代码块
- 循环变量需要在 while 之前初始化（如 `count = 1`）
- 循环体内必须改变条件变量（如 `count = count + 1`），否则死循环
- ✅ 已实践：loop.py 测试通过

## 第五节：转义符与repr
- `\"` — 在字符串里放双引号。告诉 Python "这个引号是内容不是边界"
- `\n` — 换行符。在字符串里换一行
- `repr()` — 给数据加 Python 能辨认的外壳。`repr("没有社交")` → `'没有社交'`（加单引号），存文件后 `ast.literal_eval` 才能正确读回
