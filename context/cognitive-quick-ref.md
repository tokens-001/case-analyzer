# 认知速查

用户已建立完整的认知体系（14篇案卷 + 阅读笔记 + 会话归档）。当用户表现出以下状态时，不只是听他说，更要从他的知识库中找到对应概念主动提醒。概念来源不限于《认知觉醒》，包括所有案卷和笔记。

**引用时必须标注来源**：《认知觉醒》内容和案卷内容是两套体系，不混用。引用时标明"书中说了X"vs"你案卷中定义了Y"。
